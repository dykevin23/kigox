## Skill Set

```
- Native: React-Native
- Front-end: React.js (Typescirpt)
- Back-end: Next.js
- CSS: tailwindCSS or styled-components or UI library(Material-UI)
- Database: mySQL planetScale
- ORM: prisma
- auth: NextAuth.js(naver, kakao)
- image: firebase storage
```
